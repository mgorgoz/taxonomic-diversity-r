# Security Policy

## Supported versions

The latest release of **taxdiv** on CRAN, and the current development
version on the `main` branch, receive fixes. Older versions are not
maintained.

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
| \< 1.0  | No        |

## Reporting a vulnerability

taxdiv is a statistical R package with no network, authentication, or
file-execution features, so security risk is limited. If you nonetheless
discover a vulnerability (for example, unsafe handling of a crafted
input file), please report it privately rather than opening a public
issue:

- Email the maintainer: **<muratgorgoz350@gmail.com>**
- Use a subject line beginning with `[taxdiv security]`.

Please include the package version, your R version and platform, and a
minimal example that reproduces the problem. You can expect an initial
response within a couple of weeks. Once a fix is available it will be
released to CRAN and noted in `NEWS.md`.

For ordinary bugs and feature requests, please use the regular [issue
tracker](https://github.com/mgorgoz/taxonomic-diversity-r/issues)
instead.
